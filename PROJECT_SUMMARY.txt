GETTING STARTED
================================================================================

1. INSTALL DEPENDENCIES:
   cd "/sessions/quirky-blissful-darwin/mnt/SAS3000 - Work Employment Readiness Training Arc/portfolio"
   npm install --legacy-peer-deps

2. ADD YOUR FILES:
   - Profile photo: public/images/profile.jpg (320x320px recommended)
   - Resume PDF: public/KeanGenota-IT-Intern.pdf
   - Both optional; placeholders provided if not added

3. RUN DEVELOPMENT SERVER:
   npm run dev
   Open: http://localhost:3000

4. BUILD FOR PRODUCTION:
   npm run build
   npm start

5. DEPLOY TO VERCEL:
   - Push to GitHub
   - Import project on vercel.com
   - Deploy with one click
